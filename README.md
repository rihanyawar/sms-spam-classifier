This project is a Machine Learning-based SMS Spam Classifier that predicts whether a given SMS message is Spam or Ham (Not Spam). It uses Natural Language Processing (NLP) techniques for text preprocessing, TF-IDF Vectorization for feature extraction, and the Multinomial Naive Bayes algorithm for classification. The model is deployed using Streamlit, providing a simple and interactive web interface for real-time spam detection.

Technologies Used- Python, Pandas, NumPy, Scikit-learn, NLTK, Streamlit
